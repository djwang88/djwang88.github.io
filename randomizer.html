SSBM Target Test Stage Randomizer

Randomize spawn location
Randomize character/stage mappings

Because target bounds are designed with the stage's default character in mind, it's possible
to generate unwinnable character & stage combinations.

Notes:
* Generates random (fixed) target positions, with stage-specific exclusion rules to prevent collisions
* Dynamically builds a Gecko code, which can be used with Dolphin or Wii Homebrew
* Optionally places an arbitrary number of targets between 1 and 255
* Optionally randomizes spawn location from a set of fixed spawn points
* Optionally maps every character to a random stage

For help getting set up, check out the Visual Setup Guide.
