SSBM Target Test Stage Randomizer

* Generates random (fixed) target positions, with stage-specific exclusion rules to prevent collisions
* Inserts new target coordinates into a Gecko code template, which can then be used with Dolphin or Wii Homebrew

Randomize Spawn Location


Because Gecko codes are limited in length, the all stages code has been separated into two 13-stage codes.
(Dr. Mario through Ice Climbers // Kirby through Sheik (Seak))

To do an all stage run, copy both codes as "Randomizer 1" and "Randomizer 2", then enable one at a time for the codes to work.

Known issues:
* The two moving targets in Mewtwo's stage aren't affected
* Targets intentionally can generate partially inside walls
